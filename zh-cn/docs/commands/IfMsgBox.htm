IfMsgBox

检查用户在最近的 MsgBox 命令中点击了哪个按钮.

IfMsgBox, ButtonName

参数

ButtonName

下列字符串之一, 表示用户在最近的 MsgBox 命令中所点击的按钮:

  • Yes(是)
  • No(否)
  • OK(确认)
  • Cancel(取消)
  • Abort(终止)
  • Ignore(忽略)
  • Retry(重试)
  • Continue(继续) [v1.0.44.08+]
  • TryAgain(重试) [v1.0.44.08+]
  • Timeout(即单词 "timeout" 表示消息框超时)

相关

MsgBox

示例

显示一个 "是" 或 "否" 的信息框, 5 秒后自动停止执行. 用户可按 "No" 按钮立即停止执行.

MsgBox, 4, , Would you like to continue?, 5  ; 5 秒的超时时间.
IfMsgBox, No
    Return  ; 用户点击了 "No" 按钮.
IfMsgBox, Timeout
    Return ; 即超时时假定点击了 "No".
; 否则, 继续:
; ...
