Shutdown

关闭, 重启或注销系统.

Shutdown, Flag

参数

Flag

以下数字的组合(和):

  • 0 = 注销
  • 1 = 关机
  • 2 = 重启
  • 4 = 强制
  • 8 = 关闭电源

把需要的值加在一起. 例如, 要关机并关闭电源, 标志为 9(关机 + 关闭电源 = 1 + 8 = 9). 或者, 可以使用表达式, 例如可以指定 1+8.

使用 "强制" 值(4) 来强制关闭所有打开的应用程序. 它应该仅用于紧急的情况中, 因为这样可能导致打开的应用程序数据丢失.

"关闭电源" 值(8) 表示关闭系统并关闭电源.

备注

要让系统暂停或休眠, 请参阅页面底部的示例 #2.

要关闭显示器, 请参阅 PostMessage 示例 #1.

相关提示, 脚本通过 OnExit() 或 OnExit 可以检测到系统正在关闭或用户正在注销.

相关

Run, ExitApp, OnExit(), OnExit

示例

强制重新启动(重启 + 强制 = 2 + 4 = 6).

Shutdown, 6

调用 Windows API 函数 "SetSuspendState" 来让系统挂起或休眠. 注意, 第二个参数对较新的系统可能没有任何影响.

; 参数 #1: 使用 1 代替 0 来进行休眠而不是挂起.
; 参数 #2: 使用 1 代替 0 来立即挂起而不询问每个应用程序以获得许可.
; 参数 #3: 使用 1 而不是 0 来禁止所有的唤醒事件.
DllCall("PowrProf\SetSuspendState", "int", 0, "int", 0, "int", 0)
