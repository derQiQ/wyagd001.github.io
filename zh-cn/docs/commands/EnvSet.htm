EnvSet

将值写入环境包含的变量中.

EnvSet, EnvVar, Value

参数

EnvVar
要使用的环境变量的名称, 例如 "COMSPEC" 或 "PATH".
Value

要设置到目标环境变量的值.

错误处理

[v1.1.04+]: 此命令失败时会抛出异常. 想了解更多信息, 请参阅运行时错误.

如果遇到问题则 ErrorLevel 被置为 1, 否则为 0.

备注

操作系统限制每个环境变量的文本最多为 32 KB.

此命令创建或改变的环境变量只有由此脚本通过 Run 或 RunWait 运行的程序才可以读取. 请参阅环境变量了解更多细节.

此命令与 SetEnv 分别存在, 因为普通脚本变量没有存储在环境中. 这是由于性能会变差且操作系统限制环境变量大小在 32 KB 之内.

相关

EnvGet, #NoEnv, 环境变量, EnvUpdate, SetEnv, Run, RunWait

示例

将一些文字写入环境中的 AutGUI 变量中.

EnvSet, AutGUI, Some text to put in the variable.
