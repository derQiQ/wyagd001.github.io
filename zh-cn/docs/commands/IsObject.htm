IsObject(ObjectValue) [v1.0.90+]

当 ObjectValue 是对象时返回非零数字.

TrueOrFalse := IsObject(ObjectValue)

参数

ObjectValue

可以是储存在变量的, 也可以是函数返回的, 存储在另一个对象中的或直接写入的对象.
(注1: 就是任何表示对象的变量.)
(注2: 空对象也是对象, 例如: isObject({}) 将返回 1).

返回值

当 ObjectValue 为对象时返回 1, 否则返回 0.

相关

对象

示例

报告 "This is an object." 因为值是对象.

object := {key: "value"}

if IsObject(object)
    MsgBox, 是对象.
else
    MsgBox, 不是对象.
