SoundSetVolume

更改声音设备的音量设置.

SoundSetVolume NewSetting , Component, Device

参数

NewSetting

类型: 数字或字符串

介于 -100 和 100(包含的) 之间的百分数(可以为浮点数). 如果这个数字以加号或减号开头, 则将当前设置往上或往下调指示的数量. 否则, 此设置将被明确地设置为 NewSetting 表示的级别.

Component

类型: 字符串或整数

组件的显示名称和/或索引. 例如, 1, "Line in" 或 "Line in:2".

如果省略或为空, 则更改主音量设置.

有关详情, 请参阅 Component (Sound 函数).

Device

类型: 字符串或整数

设备的显示名称和/或索引. 例如, 1, "Speakers", "Speakers:2" 或 "Speakers (Example HD Audio)".

如果省略该参数, 则它默认为系统的默认回放设备(不一定是设备 1).

有关详情, 请参阅 Device (Sound 函数).

错误处理

失败时可能会抛出以下异常之一:

Error 类消息
TargetErrorDevice not found(无法找到设备)
Component not found(无法找到组件)
Component doesn't support this control type(组件不支持此控件类型)
OSErrorSystem-generated.(由系统生成)

备注

调整音量的另一种方法是让脚本发送音量控制按键来改变整个系统的主音量, 如下例所示:

Send "{Volume_Up}"  ; 增加 1 个音程的主音量(通常为 5%).
Send "{Volume_Down 3}"  ; 降低 3 个音程的主音量.

要发现系统上安装的声音设备的功能 -- 比如名称和可用的组件 -- 请运行这个声卡分析脚本.

SoundSetVolume 在更改音量级别时会尝试保持现有的平衡.

使用 SoundGetVolume 来获取当前音量设置.

相关

Sound 函数

示例

将主音量设置为 50%.

SoundSetVolume 50

将主音量增加 10%.

SoundSetVolume "+10"

将主音量减小 10%.

SoundSetVolume -10

将麦克风录音音量增加 20%.

SoundSetVolume "+20", , "Microphone"
