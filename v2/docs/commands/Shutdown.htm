Shutdown

关机, 重启或注销系统.

Shutdown Code

参数

Code

类型: 整数

下面列出了关闭代码的组合.

备注

关闭代码是下列值的组合:

注销 0
关机 1
重启 2
强制 4
断电 8
挂起/休眠 请参阅本页底部的 DllCall 示例.
关闭显示器 请参阅 SendMessage 的示例 #1.

把需要的值加在一起. 例如, 要关闭并切断电源的代码为 9(关机 + 断电 = 1 + 8 = 9).

"强制" 值(4) 强制关闭所有打开的应用程序. 它应该仅在紧急情况下使用, 因为这样可能导致任何打开的应用程序丢失数据.

"断电" 值关闭系统并切断电源.

相关说明, 脚本通过 OnExit 可以检测到系统正在关闭或用户正在注销.

相关

Run, ExitApp, OnExit

示例

#1: 强制重新启动(重启 + 强制 = 2 + 4 = 6):

Shutdown 6

#2: 调用 Windows API 函数 "SetSuspendState" 来让系统挂起或休眠.

; 参数 #1: 使用 1 代替 0 来进行休眠而不是挂起.
; 参数 #2: 使用 1 代替 0 来立即挂起而不询问每个应用程序以获得许可.
; 参数 #3: 使用 1 而不是 0 来禁止所有的唤醒事件.
DllCall("PowrProf\SetSuspendState", "Int", 0, "Int", 0, "Int", 0)
