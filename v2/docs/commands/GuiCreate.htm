GuiCreate

创建一个新的 Gui 对象, 这是创建和管理 Gui 所必需的.

GuiObj := GuiCreate(Options, Title := A_ScriptName, EventObj)

参数

Options

类型: 字符串

此参数可以包含 Gui.Opt 支持的任何选项.

Title

类型: 字符串

窗口标题. 如果省略, 则默认为 A_ScriptName 的当前值.

EventObj

类型: 对象

"事件接收器", 或绑定事件的对象. 如果指定了 EventObj, 则可以使用 OnEvent, OnNotify 和 OnCommand 来注册引发事件时要调用的 EventObj 的方法. 如果省略或为空, 则传递给 OnEvent 的 Function 参数的任何字符串都将解释为函数的名称.

返回值

类型: 对象

此函数返回对新 Gui 对象的引用. 该对象提供了用于创建和管理窗口以及创建控件的方法和属性.
