GuiFromHwnd

检索与指定的 HWND 关联的 Gui 窗口的 Gui 对象.

GuiObj := GuiFromHwnd(Hwnd , RecurseParent := false)

参数

Hwnd

类型: 整数

Gui 窗口的窗口句柄(HWND). 这样的窗口可以用 GuiCreate 创建.

RecurseParent

类型: 整数(布尔值)

如果该参数为 1(true), 则会自动搜索并检索与指定的 HWND 最接近的父级(即 GUI).

返回值

类型: 对象

此函数返回对现有 Gui 对象的引用. 该对象提供了用于创建和管理窗口以及创建控件的方法和属性.

备注

例如, 可以通过 Gui.Hwnd, WinExist 或 WinGetID 检索 GUI 窗口的 HWND.

相关

GuiCreate, Gui 对象, GuiControl 对象, GuiCtrFromHwnd, Control 类型, ListView, TreeView, Menu 对象, Control 函数, MsgBox, FileSelect, DirSelect

示例

#1

Gui := GuiCreate(, "Title of Window")
Gui.Add("Text",, "Some text to display.")
Gui.Show()

MsgBox(GuiFromHwnd(Gui.Hwnd).Title)
