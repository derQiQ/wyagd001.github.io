Integer

将数字字符串或浮点值转换为整数.

IntValue := Integer(Value)

返回值

类型: 整数

函数返回将 Value 转换为纯整数(类型名称为 "Integer") 的结果, 或 Value 本身(如果它已经是正确的类型).

备注

任何 Value 的小数部分都会被删除, 等同于 Value < 0 ? Ceil(Value) : Floor(Value).

如果值无法转换, 则抛出 TypeError.

要确定一个值是否可以转换为整数, 使用 IsNumber 函数.

Integer 实际上是一个类, 但可以作为一个函数被调用. Value is Integer 可以用来检查一个值是否是一个纯整数.
