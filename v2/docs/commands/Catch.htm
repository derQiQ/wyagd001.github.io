Catch

指定在 try 语句执行期间发生异常时要执行的代码.

Catch OutputVar
    Statement
Catch OutputVar
{
    语句
}

参数

OutputVar

类型: 变量

(可选) 用来存储异常值的变量的名称.

Statement(s)

发生异常时执行的函数或表达式.

备注

OutputVar 可以选择括在括号中, 在这种情况下, Catch 后的空格或制表符是可选的.

catch 的每次使用都必须附属于(与之关联) 它上面的 try 语句. catch 总是附属于它上面最近的未被认领的 try 语句, 除非使用区块来改变这种行为.

可选使用 One True Brace(OTB) 样式. 例如:

try {
    ...
} catch e {
    ...
}

运行时错误

try-catch 语句也可以用来处理运行时错误. 如果运行时错误或异常没有被处理, 将显示一条错误消息, 当前线程退出. 加载时错误无法处理, 因为它们发生在执行 try 语句之前.

存储在 OutputVar(如果存在) 中的值是一个异常对象.

相关
