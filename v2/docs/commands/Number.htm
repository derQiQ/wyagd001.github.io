Number

将一个数字字符串转换为一个纯整数或浮点数.

NumValue := Number(Value)

返回值

类型: 整数或浮点数

该函数返回将 Value 转换为纯整数或浮点数的结果, 如果 Value 已经是一个整数或浮点数值, 则返回其本身.

备注

如果该值不能被转换, 则抛出 TypeError.

要确定一个值是否可以转换为一个数字, 请使用 IsNumber 函数.

Number 实际上是一个类, 但可以作为一个函数被调用. Value is Number 可以用来检查一个值是否是一个纯整数.
