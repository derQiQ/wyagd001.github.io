ListVars

显示脚本的变量: 它们的名称和当前的内容.

ListVars

备注

这个命令等同于在主窗口中选择 "View->Variables" 菜单. 它可以帮助你调试脚本.

对于列表中的每个变量, 变量的名称和内容将被显示, 以及其他信息, 取决于变量包含的内容. 每个项目以回车符和换行符(`r`n) 结束, 但如果变量包含 `r`n, 则可能跨越多行.

列表中的项目采取以下形式(其中 斜体 字符为占位符):

VarName[Length of Capacity]: String
VarName: TypeName object {Info}

Capacity 为变量当前的容量, 以字符为单位, 而不是字节.

String 为变量内容的前 60 个字符.

Info 取决于对象的类型, 但目前是非常有限的.

如果在函数内使用 ListVars, 则首先列出函数的局部变量(在脚本全局变量的上方).

已知限制: 如果函数(或全局变量自身的列表) 包含超过 10,000 个变量, 则 ListVars 可能不会以准确的字母顺序显示它们; 也就是说, 某些内容可能不显示.

相关

KeyHistory, ListHotkeys, ListLines

DebugVars 脚本可用于检查和更改变量和对象的内容.
