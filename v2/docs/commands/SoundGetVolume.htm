SoundGetVolume

从声音设备检索音量设置.

Setting := SoundGetVolume(Component, Device)

参数

Component

类型: 字符串或整数

组件的显示名称和/或索引. 例如, 1, "Line in" 或 "Line in:2".

如果省略或为空, 则检索主音量设置.

有关详情, 请参阅 Component (Sound 函数).

Device

类型: 字符串或整数

设备的显示名称和/或索引. 例如, 1, "Speakers", "Speakers:2" 或 "Speakers (Example HD Audio)".

如果省略该参数, 则它默认为系统的默认回放设备(不一定是设备 1).

有关详情, 请参阅 Device (Sound 函数).

返回值

类型: 浮点数或字符串(空值)

成功时, 返回值为 0.0 到 100.0 之间的浮点数.

失败时, 返回值为空字符串.

ErrorLevel

如果函数成功, ErrorLevel 设置为 0, 否则, 它被设置为以下短语之一:

  • Device Not Found(无法找到设备)
  • Component Not Found(无法找到组件)
  • Component Doesn't Support This Control Type(组件不支持此控件类型)
  • Can't Get Current Setting(无法获取当前设置)

备注

要发现系统上安装的声音设备的功能 -- 比如名称和可用的组件 -- 请运行这个声卡分析脚本.

示例

#1

master_volume := SoundGetVolume()
MsgBox "Master volume is " master_volume " percent."

mic_volume := SoundGetVolume("Microphone")
MsgBox "Microphone listening volume is " mic_volume " percent."

mic_volume := SoundGetVolume( , "Microphone")
MsgBox "Microphone recording volume is " mic_volume " percent."
