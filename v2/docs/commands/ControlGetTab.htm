ControlGetTab

返回 SysTabControl32 中所选选项卡的位置编号.

Tab := ControlGetTab(Control , WinTitle, WinText, ExcludeTitle, ExcludeText)

参数

Control

类型: 字符串, 整数或对象

控件的 ClassNN, 文本或 HWND, 或具有 Hwnd 属性的对象. 有关详情, 请参阅 Control 参数.

WinTitle

类型: 字符串, 整数或对象

识别目标窗口的窗口标题或其他条件. 请参阅 WinTitle.

WinText

类型: 字符串

如果存在, 此参数必须是目标窗口的单个文本元素的子字符串(和内置的 Window Spy 工具显示的一样). 如果 DetectHiddenText 为 ON, 那么会检测隐藏文本元素.

ExcludeTitle

类型: 字符串

标题中含有此参数值的窗口将被排除.

ExcludeText

类型: 字符串

文本中含有此参数值的窗口将被排除.

返回值

类型: 整数

此函数返回 SysTabControl32 中所选选项卡的位置编号. 第一个选项卡是 1, 第二个选项卡是 2, 依此类推. 如果未选择任何选项卡(罕见), 则返回值为 0.

错误处理

如果遇到问题, ErrorLevel 设置为 1, 否则设置为 0.

如果检测到无效参数, 则抛出异常.

备注

要改为发现 Tab 控件中存在多少个标签(页面), 请遵循以下示例:

TabCount := SendMessage(0x1304,,, "SysTabControl321", WinTitle)  ; 0x1304 为 TCM_GETITEMCOUNT.

窗口标题和文本是区分大小写的. 除非 DetectHiddenWindows 被打开, 否则不会检测隐藏窗口.

相关

ControlSetTab, Value 属性(GuiControl 对象), Control 函数

示例

#1: 检索第一个 Tab 控件的活动标签页序号:

WhichTab := ControlGetTab("SysTabControl321", "Some Window Title")
if ErrorLevel
    MsgBox("There was a problem.")
else
    MsgBox("Tab #" WhichTab " is active.")
