DriveGetType

返回包含指定路径的驱动器类型.

Type := DriveGetType(Path)

参数

Path

类型: 字符串

驱动器包含的任意路径(也可能对 UNC 路径和映射驱动器有效).

返回值

类型: 字符串

函数返回包含指定路径的驱动器类型: Unknown, Removable, Fixed, Network, CDROM 或 RAMDisk. 如果路径无效(例如, 因为驱动器不存在), 返回值是一个空字符串.

备注

一般来说, Path 可以是任意路径. 由于 NTFS 支持挂载卷和目录连接, 具有相同驱动器号的不同路径可以产生不同的结果.
