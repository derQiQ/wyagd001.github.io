EditGetSelectedText

返回 Edit 控件中选定的文本.

Selected := EditGetSelectedText(Control , WinTitle, WinText, ExcludeTitle, ExcludeText)

参数

Control

类型: 字符串, 整数或对象

控件的 ClassNN, 文本或 HWND, 或具有 Hwnd 属性的对象. 有关详情, 请参阅 Control 参数.

WinTitle

类型: 字符串, 整数或对象

识别目标窗口的窗口标题或其他条件. 请参阅 WinTitle.

WinText

类型: 字符串

如果存在, 此参数必须是目标窗口的单个文本元素的子字符串(和内置的 Window Spy 工具显示的一样). 如果 DetectHiddenText 为 ON, 那么会检测隐藏文本元素.

ExcludeTitle

类型: 字符串

标题中含有此参数值的窗口将被排除.

ExcludeText

类型: 字符串

文本中含有此参数值的窗口将被排除.

返回值

类型: 字符串

此函数返回 Edit 控件中选定的文本. 如果没有选择文本, 则返回空字符串. 某些类型的控件, 如 RichEdit20A 在某些情况下可能无法获取到正确的文本(例如 Metapad).

错误处理

如果找不到窗口或控件, 则抛出 TargetError.

如果获取文本时出现问题, 则抛出 Error 或 OSError.

备注

窗口标题和文本是区分大小写的. 除非 DetectHiddenWindows 被打开, 否则不会检测隐藏窗口.
