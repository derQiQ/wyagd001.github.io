ControlGetClassNN

Returns the ClassNN (class name and sequence number) of the specified control.

ClassNN := ControlGetClassNN(Control , WinTitle, WinText, ExcludeTitle, ExcludeText)

参数

Control

类型: 字符串, 整数或对象

控件的 ClassNN, 文本或 HWND, 或具有 Hwnd 属性的对象. 有关详情, 请参阅 Control 参数.

WinTitle

类型: 字符串, 整数或对象

识别目标窗口的窗口标题或其他条件. 请参阅 WinTitle.

WinText

类型: 字符串

如果存在, 此参数必须是目标窗口的单个文本元素的子字符串(和内置的 Window Spy 工具显示的一样). 如果 DetectHiddenText 为 ON, 那么会检测隐藏文本元素.

ExcludeTitle

类型: 字符串

标题中含有此参数值的窗口将被排除.

ExcludeText

类型: 字符串

文本中含有此参数值的窗口将被排除.

返回值

类型: 字符串

This function returns the ClassNN (class name and sequence number) of the specified control, or an empty string if the control could not be found.

错误处理

如果检测到无效参数, 则抛出异常.

备注

A control's ClassNN is the name of its window class followed by its sequence number within the top-level window which contains it. 例如, "Edit1" is the first Edit control on a window and "Button12" is the twelth button.

A control's ClassNN can also be determined via Window Spy, MouseGetPos or WinGetControls.

Some class names include digits which are not part of the control's sequence number. 例如, "SysListView321" is the window's first ListView control, not its 321st. To retrieve the class name without the sequence number, pass the control's HWND to WinGetClass.

窗口标题和文本是区分大小写的. 除非 DetectHiddenWindows 被打开, 否则不会检测隐藏窗口.

相关

WinGetClass, WinGetControls, ClassNN property (GuiControl object), MouseGetPos, Control 函数

示例

#1: Retrieves the ClassNN of the currently focused control:

classNN := ControlGetClassNN(ControlGetFocus("A"))
