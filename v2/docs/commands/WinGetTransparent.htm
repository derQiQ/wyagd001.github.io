WinGetTransparent

返回指定窗口的透明度的等级.

Transparent := WinGetTransparent(WinTitle, WinText, ExcludeTitle, ExcludeText)

参数

WinTitle

类型: 字符串, 整数或对象

识别目标窗口的窗口标题或其他条件. 请参阅 WinTitle.

WinText

类型: 字符串

如果存在, 此参数必须是目标窗口的单个文本元素的子字符串(和内置的 Window Spy 工具显示的一样). 如果 DetectHiddenText 为 ON, 那么会检测隐藏文本元素.

ExcludeTitle

类型: 字符串

标题中含有此参数值的窗口将被排除.

ExcludeText

类型: 字符串

文本中含有此参数值的窗口将被排除.

返回值

类型: 整数

该个函数返回一个 0 到 255 之间的整数, 该整数表示指定窗口的透明度等级, 其中 0 表示不可见窗口, 而 255 表示不透明窗口(有关如何设置透明度, 请参阅 WinSetTransparent). 返回值是一个空字符串, 如果: 1) 没有匹配窗口; 2) 窗口没有设置透明度等级; 或 3) o其他情况(由操作系统行为引起) 例如设置透明后窗口被最小化, 还原和/或重设大小.

备注

鼠标光标下的窗口的 ID 可以使用 MouseGetPos 来获取.

窗口标题和文本是区分大小写的. 除非 DetectHiddenWindows 被打开, 否则不会检测隐藏窗口.

相关

WinSetTransparent, WinGetTransColor, Win 函数, Control 函数
