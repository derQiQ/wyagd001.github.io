WinGetMinMax

返回指定窗口是最大化还是最小化的状态.

MinMax := WinGetMinMax(WinTitle, WinText, ExcludeTitle, ExcludeText)

参数

WinTitle

类型: 字符串, 整数或对象

识别目标窗口的窗口标题或其他条件. 请参阅 WinTitle.

WinText

类型: 字符串

如果存在, 此参数必须是目标窗口的单个文本元素的子字符串(和内置的 Window Spy 工具显示的一样). 如果 DetectHiddenText 为 ON, 那么会检测隐藏文本元素.

ExcludeTitle

类型: 字符串

标题中含有此参数值的窗口将被排除.

ExcludeText

类型: 字符串

文本中含有此参数值的窗口将被排除.

返回值

类型: 整数

此函数返回指定窗口的当前状态:

  • -1: 窗口处于最小化状态(使用 WinRestore 可以使其还原).
  • 1: 窗口处于最大化状态(使用 WinRestore 可以使其还原).
  • 0: 窗口既不处于最小化状态也不处于最大化状态.

如果没有匹配的窗口, 则返回一个空字符串.

备注

鼠标光标下的窗口的 ID 可以使用 MouseGetPos 来获取.

窗口标题和文本是区分大小写的. 除非 DetectHiddenWindows 被打开, 否则不会检测隐藏窗口.
