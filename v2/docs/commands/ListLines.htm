ListLines

启用或禁用行日志记录或显示最近执行的脚本行.

ListLines OnOff

参数

OnOff

类型: 字符串

如果为空或省略, 则显示最近执行的脚本行的历史. 第一个参数仅影响当前线程的行为, 如下所示:

On 或 1(true): Includes subsequently-executed lines in the history. This is the starting default for all scripts.

Off 或 0(false): Omits subsequently-executed lines from the history.

备注

ListLines (with no parameter) is equivalent to selecting the "View->Lines most recently executed" menu item in the main window. It can help debug a script.

ListLines OnOff can be used to selectively omit some lines from the history, which can help prevent the history from filling up too quickly (such as in a loop with many fast iterations). Additionally, performance is reduced by a few percent while line logging is enabled.

每个新运行的线程(如 热键, 自定义菜单项或定时子程序) 都会以此函数的默认设置开始. 这个默认设置可以通过在脚本的自动执行段(脚本的顶部) 使用此函数来改变.

The built-in variable A_ListLines contains 1 if ListLines is enabled and 0 otherwise.

On a related note, the built-in variables A_LineNumber and A_LineFile contain the currently executing line number and the file name to which it belongs.

相关

示例

#1

ListLines
ListLines "Off"
ListLines False
