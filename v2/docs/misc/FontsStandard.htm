标准 Windows 字体

用于 Gui.SetFont. "开始存在于" 列代表首次包含该字体的 Windows 版本. 表格中的常见的等宽字体有 Consolas, Courier, Courier New, Fixedsys, Lucida Console 和 Terminal. 某些字体的可用性可能取决于操作系统的语言.

黄色高亮显示的为推荐使用的字体. 请注意, 这里推荐的字体是自 Windows XP 或更早版本以来可读和系统已经包含的字体. 使用这种字体可确保在大多数操作系统上显示它们.

注意: 这些建议还没有对 AutoHotkey v2.0 进行审核, 其需要 Windows Vista 或更高版本.

点击列标题可对表格的内容进行排序.
